-- Actions Resources --

Get all actions.
GET /application/actions
< 200
< Content-Type: application/json
{ }

Add a new action.
POST /application/actions
> Accept: application/json
{ "nameaction": "giveComment", "description": "Mettre un commentaire" }
< 201
< Content-Type: application/json
{ }


Get an application user through his id.
GET /application/actions/{id}
> Accept: application/json
{ « id »: « 1 » }
< 200
< Content-Type: application/json
{ }

Update an application user through is id and a new representation.
PUT /application/actions/{id}
> Accept: application/json
{ « id »: « 1 », « nameaction »: « toto », « description »: « Comms »}
< 204
{}

Delete an action by passing its ID
DELETE /application/actions/{id}
< 200
< Content-Type: application/json
{}
